<div align="center">

**Kuaishou NVC** is a neural video coding framework developed by Kuaishou, pursuing high compression performance.

*This repository provides access to code, pretrained models, and research papers for various versions of the Kuaishou NVC codec.*

## üì∞ **Release Notes**

#### **BRHVC** - **First Release**
- **Release Date**: *Jan 2026*
- **Codec Version**: BRHVC (Bi-directional Reference Harmonization Video Compression)
- **Key Features**:
  - This version implements the core concepts and methods from the paper **"Neural B-frame Video Compression with Bi-directional Reference Harmonization"**, accepted at NeurIPS 2025.
